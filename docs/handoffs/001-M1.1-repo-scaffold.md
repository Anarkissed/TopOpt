## Task
Repo scaffold: CMake project, `core/` layout per ARCHITECTURE §3, CI workflow that builds and runs an empty ctest suite on Linux with OCCT + Eigen + lib3mf installed. Green pipeline is the deliverable.

## What I did
- Moved `ARCHITECTURE.md`, `ROADMAP.md`, `AGENT_PROMPTS.md` from the repo root into `docs/` (git-detected renames, zero content changes — they predated the §3 layout). Created `docs/DECISIONS.md` as an empty human-owned stub and `docs/handoffs/`.
- Created the full §3 directory tree: `core/include/topopt/`, `core/src/{io,mesh,voxel,fea,simp,orient,settings,materials}/`, `core/tests/{unit,validation,property,fixtures}/`, `app/`, `ci/`. Empty directories hold a `.gitkeep` only. `app/` was created as an empty directory (layout only — no app work).
- `core/CMakeLists.txt`: C++17 `topopt` library target. Its only translation unit is `src/version.cpp` (a `topopt::version()` returning "0.1.0" — a library target needs one symbol; no other API). `enable_testing()` with zero registered tests.
- Dependency verification is behind `option(TOPOPT_REQUIRE_DEPS)` (default OFF): when ON, `find_package` for `OpenCASCADE`, `Eigen3`, `lib3mf` are `REQUIRED`. CI turns it ON so the pipeline proves the three locked dependencies are installed and findable; local dev machines without them can still configure/build/test.
- Dependencies come from **vcpkg** (manifest `core/vcpkg.json`, baseline pinned to release 2026.06.24, commit `cd61e1e2`), not apt. Why: ubuntu-latest apt has OCCT 7.6.x, below the ≥ 7.8 locked in ARCHITECTURE §4, and lib3mf is not in apt at all. At the pinned baseline vcpkg provides opencascade 8.0.0, lib3mf 2.5.0, eigen3 5.0.1. Triplet is `x64-linux-dynamic` so OCCT stays dynamically linked (LGPL 2.1, §4/§10).
- CI: canonical workflow at `.github/workflows/ci.yml`; `ci/ci.yml` is a symlink to it. GitHub Actions only executes workflow files physically located in `.github/workflows/` (symlinked workflows are ignored), so the "ci/ci.yml referenced from .github/workflows/" wording had to be implemented in this direction. Job: ubuntu-latest → apt build tools + X11/GL headers → pinned vcpkg clone → `actions/cache` on the vcpkg binary-archive dir → configure (`-DTOPOPT_REQUIRE_DEPS=ON`) → build → `ctest --output-on-failure --no-tests=ignore`.
- Added minimal `README.md` and `.gitignore`.
